# ember-cli-tbs

Ember CLI Addon for Twitter Bootstrap.

## Installation

* `ember install:addon ember-cli-tbs`

## Running

* `ember server`
* Visit your app at http://localhost:4200.

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`

For more information on using ember-cli, visit [http://www.ember-cli.com/](http://www.ember-cli.com/).
